# fraction_vis

## Deploy

The GitHub remote is `https://github.com/RupertLinacre/fraction_vis.git`.

This project is served at `https://rupertlinacre.com/fraction_vis`, so the
GitHub Pages build uses Vite's `/fraction_vis/` base path.

```sh
npm run deploy
```

That builds the app with `npm run build:pages` and publishes `dist/` to the
`gh-pages` branch.
